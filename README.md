# Blog Application 

Stack used: Ruby On Rail & Bulma

## Features!

  - Post article 
  - Edit and Update article
  - Delete article
  - Home page showing articles
  - Comment on articles
  - Shows number of comments made

## Upcoming Featrures

  - Logging and Signup System
